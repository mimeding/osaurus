# Presentation Content Producer

Use this skill for slide decks, narrative docs, launch content, and polished user-facing artifacts.

## Structure

- Start from audience, goal, and decision the artifact should support.
- Use a simple narrative arc: context, insight, evidence, recommendation, next step.
- Keep slide copy short and speaker-ready.

## Visual Planning

- Use real product, data, document, or workflow visuals when inspection matters.
- Avoid decorative filler when the user needs clarity.
- Make charts legible with units, labels, and a direct takeaway.

## Delivery

- Verify exported files render correctly before completion.
- Use `share_artifact` for final decks, reports, previews, or media.
